前 言

 

《算法设计与分析A》是一门面向设计，处于计算机科学与技术学科核心地位的教育课程。通过对计算机算法系统的学习，使学生理解和掌握计算机算法的通用设计方法，培养对算法的计算复杂性正确分析的能力，为独立设计算法和对算法进行复杂性分析奠定基础。

本书是本科课程《算法设计与分析A》的实验指示书，使用它作为学习该课程的上机指导。通过本课程的实验，使学生加深对课程内容的理解，培养学生严密的思维能力，运用所学知识结合具体问题设计适用的算法的能力；培养学生良好的设计风格，激励学生创造新算法和改进旧算法的愿望和热情。希望同学们能够充分利用实验条件，认真完成实验，从实验中得到应有的锻炼和培养。

在使用本实验指导书及进行实验的过程中，希望同学们能够帮助我们不断地发现问题，并提出建议，使《算法设计与分析A》课程成为对大家有益的课程。

procedure Knapsack(p, w, M,n)

//其中M存背包总的容量，p[n]存各物品的效益值，w[n]存各物品的容量，m[i][j]含义见上述分析；

real p[n],;

int w[n],n,M;

int jMax=min(w[n]-1,M);

real m[n][jMax]

for jß0 to jMax do

   m[n][j]=0;

repeat

for jßw[n] to M do

      m[n][j]=p[n];

repeat

for ißn-1 to 1 step -1do

  jMax=min(w[i]-1,M);

  for jß0 to jMax  do

m[i][j]=m[i+1][j];

         repeat

      for jßw[i] to M  do

m[i][j]=max(m[i+1][j],m[i+1][j-w[i]]+p[i]);

repeat

       repeat

m[1][M]=m[2][M];

if M>=w[1]

        then m[1][M]=max(m[1][M],m[2][M-w[1]]+p[1]);

endif

endKnapsack

 

procedure Traceback(m,w, M,n)

real,m[n][n];

int w[n],n,M ,x[n];

    for iß1 to n  do

         if m[i][M]==m[i+1][M]

then  x[i]=0;

else x[i]=1;M=M-w[i];

         endif

repeat

x[n]=m[n][M])?1:0;

end Trackback
